# Chat UI Project

This project is a simple user interface similar to ChatGPT, built with HTML, CSS, and JavaScript. It allows users to interact through a chat interface, sending and receiving messages.

## Project Structure

```
chat-ui-project
├── src
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   └── index.html
├── package.json
└── README.md
```

## Features

- User-friendly chat interface
- Input fields for user messages
- Display area for conversation history
- Responsive design

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd chat-ui-project
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in a web browser to view the application.

## Usage

- Type your message in the input field and press "Send" to communicate.
- The chat history will display above the input field.
